# Future Ideas

- Richer media preview, especially video preview and stronger document preview support.
- On `Enter` for a file, open it with the default application registered by Windows.
- On file right-click, show the Windows-registered "Open with" program list and let the user choose which app to launch.
- In the terminal, make `Tab` behave like a normal shell autocomplete for files/paths instead of moving focus back to the file panes.
